Copies the data from the pointer pointing to a part in the active heap to the first empty slot in the passive part. Will also mark the header of the object in the active part as copied to the passive part. NOTE: This does not alter the data in any way, it is a clean copy of the data.

Parameters
heap- The heap to work on.
data- Pointer pointing to the data to copy over.
Returns
Returns pointer to the copied data. This will still be a legal pointer after a call to heap_swapActiveAndPassive function.
